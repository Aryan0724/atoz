Getting Started

How do I set up my print files? expand_more
We recommend using high-resolution PDF files (300 DPI) with CMYK color profiles. Ensure all fonts are outlined and include a 3mm bleed on all sides for optimal results.
What file formats do you accept? expand_more
We primarily accept .PDF, .AI, .PSD, and .TIFF files. For small format products, high-quality .JPEG and .PNG are also acceptable.

Shipping & Logistics

Do you ship internationally? expand_more
